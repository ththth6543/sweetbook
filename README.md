# ✈️ Travel Book (트래블북)

> **"당신의 소중한 여행 기록, 이제 손에 잡히는 추억으로 간직하세요."**
> 

## 1. 서비스 소개

### 📖 한 줄 설명

사용자의 여행 일지(로그)와 사진을 세련된 템플릿에 맞춰 하나의 **실물 포토북으로 자동 생성 및 주문**할 수 있도록 돕는 서비스입니다.

### 🎯 타겟 고객

- 디지털 사진을 실물 앨범으로 소장하고 싶은 여행자
- 여행의 추억을 부모님이나 친구에게 특별한 선물로 전하고 싶은 분
- 복잡한 편집 과정 없이 고품질의 사진책을 갖고 싶은 사용자

### ✨ 주요 기능

- **여행 로그 관리**: 여행별 글과 사진을 기록하고 리스트 형태로 체계적으로 관리합니다.
- **템플릿 기반 생성**: 스위트북(Sweetbook) API를 연동하여 다양한 판형과 전문적인 디자인 템플릿을 제공합니다.
- **스마트 편집**: 페이지별 파라미터 최적화 및 사진 업로드 기능을 통해 맞춤형 내지를 구성합니다.
- **주문 및 크레딧 시스템**: 제작 견적 조회부터 샌드박스 환경을 통한 가상 결제 및 주문 프로세스를 지원합니다.

## 3. 핵심 API 명세 (Sweetbook API 연동)

| **분류** | **메서드** | **엔드포인트** | **설명** |
| --- | --- | --- | --- |
| **스펙/템플릿** | `GET` | `/book-specs` | 도서 판형 정보 조회 |
|  | `GET` | `/templates` | 구역별(표지, 내지) 템플릿 목록 조회 |
| **책 편집** | `POST` | `/books` | 신규 포토북 프로젝트 생성 |
|  | `POST` | `/books/{uid}/photos` | 템플릿 삽입용 이미지 업로드 |
|  | `POST` | `/books/{uid}/cover` | 표지 템플릿 및 파라미터 적용 |
| **주문/결제** | `POST` | `/orders/estimate` | 제작 예상 비용(견적) 조회 |
|  | `POST` | `/orders` | 최종 제작 및 배송 주문 |
| **샌드박스** | `POST` | `/credits/sandbox/charge` | 테스트용 가상 크레딧 충전 |

## 4. AI 협업 및 설계 의도

### 🤖 AI (Antigravity, Gemini) 활용 내역

- **아키텍처 설계**: Flutter Riverpod 기반의 상태 관리와 FastAPI의 비동기 통신 구조 스캐폴딩.
- **데이터 모델링**: Sweetbook API의 복잡한 JSON 응답을 Dart 클래스로 자동 변환 및 역직렬화 처리.
- **디버깅**: 샌드박스 API 호출 시 발생하는 파라미터 불일치 및 Null Safety 오류 신속 해결.

### 💡 기획 의도 및 비즈니스 가치

- **감성적 가치**: 클라우드에 잠들어 있는 디지털 데이터를 '손에 잡히는 물성'으로 전환하여 사용자 경험의 완성을 추구합니다.
- 레코드판, CD 등을 수집하듯이 나의 혹은 친구들과의 여행을 수집하는 모습을 생각하며 구상하였습니다.
- **확장 가능성**:
    - 개인 여행 굿즈 시장 공략.
    - 여행사 패키지 상품의 포함 서비스 또는 크리에이터 한정판 굿즈 제작 플랫폼으로 확장 가능.

### 🚀 추가하고 싶은 기능들

1. **AI 자동 레이아웃**: 사진의 메타데이터(날짜/장소)를 분석해 최적의 순서로 자동 배치.
2. **지도 연동 페이지**: 여행 위치를 국내면 카카오 지도, 네이버 맵을 국외면 구글 맵을 이용하여 특정위치를 QR코드로 담아서 보여주는 기능
3. **실시간 알림**: 웹훅을 통한 제작 및 배송 상태 푸시 알림.
4. 페이지 관련해서 완전한 알고리즘을 담지 못해서 아쉬움
